# IELTS Vocab - 雅思单词记忆App 需求文档

## 1. 项目概述

### 1.1 产品简介
IELTS Vocab 是一款面向雅思考生的单词记忆移动应用，基于间隔重复算法，结合多种记忆方法，帮助用户高效掌握雅思核心词汇。

### 1.2 目标用户
正在备考雅思的学生群体，需要高效记忆雅思核心词汇。

### 1.3 核心问题
- 雅思词汇量大（约8000+），传统死记硬背效率低
- 缺乏科学的复习计划，学完容易忘
- 单一记忆方式难以应对不同类型单词

## 2. 核心功能

### 2.1 智能复习提醒
- 基于艾宾浩斯遗忘曲线的间隔重复算法（SM-2）
- 每日自动生成复习计划，根据掌握程度调整复习频率
- 每日打卡机制，连续学习天数统计
- 本地推送通知提醒用户按时复习

### 2.2 多种记忆方法
每个单词支持以下记忆辅助：
- **词根词缀拆解**：展示单词的词根、前缀、后缀及含义
- **联想记忆**：提供助记口诀或联想故事
- **语境例句**：展示雅思真题或地道用法例句
- **图片辅助**：对具象名词配图增强记忆
- **发音播放**：英式/美式发音，支持跟读

### 2.3 学习数据统计
- 单词掌握度分布（新学 / 模糊 / 已掌握）
- 学习天数与连续打卡天数
- 每日/每周学习量趋势图
- 正确率统计
- 词汇量增长曲线

### 2.4 词汇分类浏览
- 按雅思话题分类（教育、环境、科技、健康等）
- 按难度分级（基础 / 核心 / 进阶）
- 按词性分类
- 搜索与收藏功能
- 雅思高频词优先推荐

## 3. 页面结构

### 3.1 导航方案
底部 Tab 导航，4个主Tab：

| Tab | 页面 | 功能 |
|-----|------|------|
| 学习 | 单词学习页 | 卡片式学单词，当日学习+复习任务 |
| 词库 | 词汇浏览页 | 分类浏览、搜索、收藏列表 |
| 统计 | 数据统计页 | 学习数据可视化、打卡日历 |
| 我的 | 个人中心页 | 设置、学习偏好、数据备份 |

### 3.2 学习页（核心页面）
- 顶部：当日任务概览（新学N个 / 复习N个）
- 中部：单卡片沉浸式展示单词
  - 正面：英文单词 + 音标 + 发音按钮
  - 背面（点击翻转）：中文释义 + 词性 + 记忆方法 + 例句
- 底部：三个操作按钮（不认识 / 模糊 / 认识）
- 进度条显示当日完成进度

### 3.3 词库页
- 顶部搜索栏
- 分类标签（话题/难度/词性）
- 单词列表，支持收藏标记
- 点击进入单词详情页

### 3.4 统计页
- 学习概览卡片（总词汇量、今日新学、连续天数）
- 掌握度环形图
- 周学习量柱状图
- 打卡日历

### 3.5 个人中心页
- 用户头像与昵称
- 每日学习目标设置
- 学习提醒时间设置
- 数据备份与恢复
- 关于与反馈

## 4. UI 设计规范

### 4.1 设计风格
柔和渐变风（Soft UI），现代优雅，适合沉浸式学习体验。

### 4.2 配色方案
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 紫蓝渐变 | #6366F1 → #8B5CF6 |
| 背景色 | 淡紫灰 | #F8FAFC |
| 文字主色 | 深紫黑 | #1E1B4B |
| 文字辅色 | 中灰 | #6B7280 |
| 强调色 | 柔紫 | #818CF8 |
| 边框色 | 淡紫边框 | rgba(99,102,241,0.1) |
| 成功色 | 绿 | #16A34A / #DCFCE7 |
| 警告色 | 橙 | #EA580C |

### 4.3 字体
- 全局字体：Plus Jakarta Sans
- 标题：Bold (700)
- 正文：Regular (400)
- 辅助文字：Light (300)

### 4.4 设计要素
- 圆角：卡片 14-16px，按钮 8-10px，标签 4-6px
- 阴影：柔和微阴影，`0 2px 8px rgba(0,0,0,0.04)`
- 渐变：紫蓝渐变用于进度条、按钮等强调元素
- 间距：基于 4px 网格系统

## 5. 技术栈

### 5.1 前端
- **框架**：Flutter (Dart)
- **状态管理**：Riverpod
- **本地数据库**：Isar / Hive
- **图表**：fl_chart

### 5.2 数据
- 雅思词汇数据集（本地 JSON 打包）
- 单词学习进度（本地存储）
- 用户设置与偏好（本地存储）

### 5.3 离线优先
- 核心功能完全离线可用
- 无需服务端，数据全部本地存储
- 可选：未来支持云端同步备份

## 6. 数据模型（核心）

```
Word {
  id: String
  word: String          // 英文单词
  phonetic: String      // 音标
  meanings: [{          // 释义列表
    partOfSpeech: String  // 词性
    definition: String    // 中文释义
  }]
  examples: [String]    // 例句
  roots: String         // 词根词缀
  mnemonic: String      // 联想记忆
  topic: String         // 话题分类
  level: String         // 难度等级
  frequency: Int        // 频率排名
}

UserProgress {
  wordId: String
  ease: Double          // SM-2 难度因子
  interval: Int         // 复习间隔（天）
  repetitions: Int      // 连续正确次数
  nextReview: DateTime  // 下次复习日期
  lastReview: DateTime  // 上次复习日期
}

UserSettings {
  dailyNewWords: Int    // 每日新学数量
  reminderTime: Time    // 提醒时间
  soundEnabled: Bool    // 发音开关
}
```

## 7. 非功能需求

- 启动时间 < 2秒
- 本地数据操作响应 < 100ms
- 支持 iOS 14+ 和 Android 8+
- 适配不同屏幕尺寸
